# 环境要求

---

* NodeJS\(v4.0.0及以上\)

#  通过NPM安装

---

运行下面的命令进行安装

```
npm install gitbook-cli -g
```

其中`gitbook-cli`是gitbook的一个命令行工具, 通过它可以在电脑上安装和管理gitbook的多个版本.
